# nest-test

## Requirements

- Node.js >= 22.19.0

## Installation

```bash
npm install
```

---

# Compile and run the project

### development

```bash
npm run start
```

### watch mode

```bash
npm run start:dev
```

### production mode

```bash
npm run start:prod
```

## Run tests

### unit tests

```bash
npm run test
```

### e2e tests

```bash
npm run test:e2e
```

### test coverage

```bash
npm run test:cov
```
